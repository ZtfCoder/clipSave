# Product Overview

ClipboardManager is a native macOS clipboard history manager built with SwiftUI.

## Core Features
- Automatic clipboard history tracking (text, images, files)
- Global hotkey activation (⌘⇧V)
- Menu bar integration with popover UI
- Floating panel at cursor position
- Search/filter through history
- Auto-paste to previous application
- Local persistence via UserDefaults
- Launch at login support

## Target Platform
- macOS 13.0 (Ventura) or later
- Runs as menu bar accessory app (no Dock icon)

## Permissions Required
- Accessibility permission for global hotkeys and simulated paste
